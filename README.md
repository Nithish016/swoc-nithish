# Plant Disease Classification

An advanced Flask-based web application for classifying plant leaf diseases using customized Deep Learning models based on convolutional neural networks.

## Features
* Upload leaf images for inference.
* Classify images across extensive categories containing multiple plant species and disease states.
* Supports customized architectures.
* Dynamic web user interface built with Bootstrap and modern design elements.

## Running the Project Local Application

### Requirements
You can review all required dependencies in `requirements.txt`. Make sure you have Python 3.10 or 3.11/3.12 (for TensorFlow & PyTorch compatibility).

1. Clone or download this project
2. Activate a Python virtual environment
3. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the Flask application:
   ```bash
   python app.py
   ```
5. Navigate to `http://127.0.0.1:5000` in your web browser.
